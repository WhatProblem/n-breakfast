### n-breakfast

> nuxt backend for breakfast shop

### Build Setup

```shell
# 安装依赖
npm install --registry=https://registry.npm.taobao.org

# 运行本地服务 localhost:3000
npm run dev

# 构建生产环境，然后运行
npm run build
npm run start
```

一个商城的后台管理网站，后续会陆续完善...
